# GuideMe Ecosystem - Vision & Roadmap

## üéØ Vision

**Current:** A browser extension that provides AI-powered step-by-step guidance for any website.

**Goal:** Expand to desktop application software (Adobe, Figma, Blender, etc.) - because new software and websites emerge every day, and people will always need help navigating them.

## ‚úÖ What We've Built (Completed)

### Core Extension
- [x] AI-powered guidance using Gemini/OpenAI/Claude
- [x] Multi-page flow handling with state persistence
- [x] Voice input support
- [x] Robust element identification with multiple fallback selectors
- [x] SPA (Single Page App) support

### Guide Management
- [x] Save, rename, delete guides
- [x] Categories and search/filter
- [x] `.guideme` file format with SHA-256 checksums

### UI/UX
- [x] Professional design with CSS design system
- [x] SVG icon system (no emojis)
- [x] Scrollable guide lists
- [x] Smooth animations

### E-commerce & Finance
